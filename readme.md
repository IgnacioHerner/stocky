# Stocky

Stocky es una app Android en desarrollo para emprendedores, enfocada en
gestión de inventario y ventas.

## Objetivo
Permitir registrar productos, stock y ventas, y visualizar métricas
básicas del negocio.

## Stack
- Kotlin
- Jetpack Compose
- Room
- Coroutines / Flow
- MVVM

## Estado
Proyecto en desarrollo (MVP).
